Houses the CLC Web Services PHP libraries and manages access keys. Required by another CLC plugin.

== Description ==

This plugin is required by another plugin, which uses its libraries and its settings to connect to CLC Object Storage. Currently, there is only one plugin that requires this plugin:

* CLC Object Storage

== Installation ==

1. Use WordPress' built-in installer
2. A new CLC Object Storage menu will appear in the side menu
